# Malaysian Institutional Fintech & Compliance Gateway AI

Combines high-level regulatory RAG with survival tools and market analysis to create a "one-stop" platform for Malaysian financial technology.  

# Core Components:
• RegComply AI: Acts as the primary intelligence layer for BNM/SC guidelines.  
• Survival Pro: Provides the user-facing B40 financial aid and Spend-Coach features.  
• FXWatch: Supplies real-time corridor monitoring for MYR/SGD/USD.  
• Bursa Factor Risk Model: Adds institutional-grade risk decomposition for local portfolios.
