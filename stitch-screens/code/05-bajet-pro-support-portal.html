Terima kasih Encik Faris. Error code FPX_ERR_4021 bermaksud "Bank Account Verification Timeout" — ini isu teknikal di pihak CIMB, bukan masalah Bajet Pro.

Saya telah log isu ini kepada pasukan teknikal kami dan juga BNM Payment Gateway team. Sementara itu, anda boleh cuba:

✅ Gunakan Maybank2u atau RHB Now sebagai alternatif sementara
✅ Cuba semula antara 5:00 PM – 8:00 PM (biasanya server FPX CIMB lebih stabil)

Saya akan update awak dalam masa 2 jam.
